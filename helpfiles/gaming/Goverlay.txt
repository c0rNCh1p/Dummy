GOVERLAY

DESCRIPTION
GOverlay is an open source project created by Benjamimgois and Flightlessmango
aimed to create a Graphical UI to manage Vulkan and OpenGL overlays with a
preview. It now supports the management of a range of different gaming utilities
including Flightlessmanogs MangoHud, vkBasalt and ReplaySorcery.
